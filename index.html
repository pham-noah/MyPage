HTML

The <html> element is the root element of an HTML page
It has a start tag <html> and an end tag </html>.

Head

The <head> element contains metadate (data about data) information about the HTML page. And it is placed between the <html> tag and the <body> tag.

Body

The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

It has a start tag <body> and an end tag </body>.

Paragraph

The <p> element defines a paragraph
It has a start tag <p> and an end tag </p>
Note: A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Images

Images can improve the design and the appearance of a web page.
HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provided as attributes.
The <img> tag is empty, it contains attributes only, and does not have a closing tag.

HMTL Links

HTML links are hyperlinks.
You can click on a link and jump to another document.
When you move the mouse over a link, the mouse arrow will turn into a little hand.

HTML links are defined with the <a> tag
The link's destination is specified in the href attribute.
Attributes are used to provide additional information about HTML elements.

Breaks

The <br> tag defines a line break, and is an empty element without a closing tag

Title

The <title> element:

  • <defines a title in the browser toolbar>
  • <provides a title for the page when it is added to favorites >
  • <displays a title for the page in search engine-results >
HTML Headings

HTML headings are defined with the <h1> to <h6> tags.

Heading Type Function
<h1> defines the most important heading.
<h6> defines the least important heading
